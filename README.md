##  研究内容
ロールモデル指向の対話エージェントの影響分析：

調査の対象：ロールモデル（特定の職業やキャラクター）を模倣するChat-GPTの効果。
目的：特定のロールモデルを持つChat-GPTがユーザーの知識獲得、意思決定、満足度に与える影響を分析。
方法：①「異なるロールモデル」、
②異なるロールモデルの指示方法（Prompt）
をした際のChat-GPTの応答を収集し、ユーザーの反応や評価を調査。
